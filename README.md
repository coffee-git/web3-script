# web3-script

币圈交互系列脚本

本系列脚本分为 
- crypt-module(加解密模块)
- utils-module(工具库模块)
- exchange-script(交易所脚本)
- utxo-script(utxo系交互脚本)
- evm-script(evm系交互脚本)
- sol-script(sol链交互脚本)
- sui-script(sui链交互脚本)

持续完善中。。。

## 免责声明

本脚本只作为学习用途，使用风险自负！

#### 配置

`.env-example`是示例配置文件，实际使用中需要同目录下创建`.env`文件，内容根据实际情况修改。用到哪个模块可查看模块文档增加配置。

#### 启动脚本
```
npm run dev
```

#### 浏览器交互
```
http://localhost:3003/
```
